# Michaella Food

Bienvenue dans le projet **Michaella Food** !

## À propos du projet

Michaella Food est un projet de site web pour un restaurant fictif. Il a été créé dans le but de démontrer les compétences en développement web et d'explorer diverses fonctionnalités, y compris la possibilité de traduire le contenu en plusieurs langues.

## Fonctionnalités

- Présentation d'un menu de restaurant
- Informations sur les événements spéciaux
- Présentation de l'équipe de chefs
- Galerie de photos
- Fonctionnalité de traduction pour le contenu en français et en anglais

## Comment contribuer

Si vous souhaitez contribuer à l'amélioration de ce projet, n'hésitez pas à soumettre des pull requests ou à ouvrir des issues.
